TRINITY CLG WEBSITE
A simple and dynamic college website built using HTML,CSS and javascript
OVERVIEW:This project is a website for TRINITY COLLEGE.designed to provide students,faculty and visitors with essential information about the college.the site includes an interactive interface,coufrse details ,branch
          information,notes for students and a direct link to chech results
          
FEATURES:
         HOME PAGE:Displays a welcoming message and credits the creator
         COURSES OFFERED1:Lists courses like MBA,B.TECH,DIPLOMA,B.ED, and B.PHARMACY.
         BRANCHES:shows specific branches under each course
         NOTES SECTION:provides study materials and notes categorized by course and branch.
         RESULT PAGE:directly links to a google based results portal for easy access.

TECHNOLOGIES USED:
                  FRONTEND:html,css,js

    navigate thriugh the me nu to explore courses ,branches,notes,and results more efficiently.
